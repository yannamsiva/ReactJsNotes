React Component Life Cycle
==========================
->React Js every component creation process involves various lifecycle methods.
->These methods are termed as components lifecycle methods The lifecycle of the component is divided into four phases.
-----------------------------------------------------------
1.Initial phase
  ->getDefaultProps()
  ->getInitialState() 2.Mounting Phase
  ->componentWillMount()
  ->componentDidMount()
  ->render() 3.Updating Phase
  ->componentWillREcieveProps();
  ->shouldComponentUpdate();
  ->componentWillUpdate()
  ->render()
  ->componentDidUpdate(); 4.Unmounting Phase
  ->componentWillUnmount();  Each phase contains some lifecycle methods React Js Class Component LifeCycle Application Development Steps
